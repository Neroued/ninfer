# Pre-Optimization Documentation

This archive contains the foundation-stage material written before the main kernel-optimization
campaign: L0/L1/L2 plans, M2 hardening, M2.8 stage-gate documents, early workflow notes, and their
generated plans/specs.

These files are historical records. Their status labels, source paths, commands, format references,
and “current document” links describe the repository at the time and are not maintained. Use the
active documentation index at [`../../README.md`](../../README.md) for the implemented system.

## Contents

- `plans/` — completed L0, L1, L2, M2.5, and M2.8 plans;
- `superpowers/plans/` and `superpowers/specs/` — completed generated plans and design specs;
- `m2.8-pre-m3-standard*.md` — the historical readiness standard;
- `l1-implementation-roadmap.md` and `l1-gdn-port-design.md` — early implementation design;
- `chatgpt-web-workflow.md` — the historical planning workflow.

Preserving these documents does not preserve the old APIs, q5090 formats, test structure, or command
surface they describe.

# M2.8 P2 Engine Memory Observability Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development
> (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox
> (`- [ ]`) syntax for tracking.

**Goal:** Expose dependency-free Engine memory stats and arena high-water marks so e2e reports can record
weight/cache/workspace capacity, used bytes, and repeat-local peaks.

**Architecture:** Add high-water accounting inside `DeviceArena`, then aggregate existing arena and
`WeightStore` state through `Engine::memory_stats()`. Timing remains in benchmark code; P2 adds no JSON
writer, no loader redesign, and no process/RSS/CUDA-runtime memory accounting.

**Tech Stack:** C++17, CUDA runtime, CMake/CTest, current single-file test executables under `tests/`.

---

## Scope

P2 changes code, tests, and plan-status documentation.

P2 owns:

- `DeviceArena::peak_used()`.
- `DeviceArena::reset_peak()`.
- high-water semantics across `alloc`, `rewind`, `reset`, failed allocation, and moves.
- verification that existing `WeightStore` q5090 counters remain usable by Engine reporting.
- `ArenaMemoryStats`.
- `EngineMemoryStats`.
- `Engine::memory_stats()`.
- `Engine::reset_memory_peaks()`.
- tests for all of the above.

P2 does not own:

- e2e benchmark timing.
- JSON report writing.
- report comparison.
- q5090 loader redesign.
- q5090 real-weight regeneration.
- process RSS, CUDA runtime internals, or profiler overhead accounting.
- workspace lifetime changes.
- P1 conv1d layout implementation.

P2 is sequenced after P1 for this implementation pass. Execute it directly in the current workspace after
P1 is committed and the current workspace is clean; no separate worktree is required.

## Required Public Contract

The final public API must match the M2.8 standard:

```cpp
std::size_t DeviceArena::used() const noexcept;
std::size_t DeviceArena::capacity() const noexcept;
std::size_t DeviceArena::peak_used() const noexcept;
void DeviceArena::reset_peak() noexcept;
```

```cpp
struct ArenaMemoryStats {
    bool present = false;
    std::size_t capacity_bytes = 0;
    std::size_t used_bytes = 0;
    std::size_t peak_used_bytes = 0;
};

struct EngineMemoryStats {
    bool loaded = false;
    int device = 0;
    std::uint32_t max_context = 0;
    std::uint32_t position = 0;
    ArenaMemoryStats weights;
    ArenaMemoryStats cache;
    ArenaMemoryStats workspace;
    std::size_t q5090_loaded_payload_bytes = 0;
    std::size_t q5090_tensor_count = 0;
    std::size_t q5090_quant_count = 0;
};

EngineMemoryStats Engine::memory_stats() const noexcept;
void Engine::reset_memory_peaks() noexcept;
```

`memory_stats()` must not throw, synchronize, or initialize CUDA by itself. Before load, it must report
`loaded == false`, absent arenas, zero memory counters, `device == options.device`,
`max_context == options.max_ctx`, and `position == 0`.

namespace {

constexpr std::size_t kMiB = 1024ULL * 1024ULL;

bool cuda_unavailable(cudaError_t err) {
    return err == cudaErrorNoDevice || err == cudaErrorInsufficientDriver;
}

int fail(std::string_view message) {
    std::cerr << message << '\n';
    return 1;
}

int expect_size(std::size_t actual, std::size_t expected, std::string_view label) {
    if (actual == expected) { return 0; }
    std::cerr << label << " expected " << expected << ", got " << actual << '\n';
    return 1;
}

int expect_u32(std::uint32_t actual, std::uint32_t expected, std::string_view label) {
    if (actual == expected) { return 0; }
    std::cerr << label << " expected " << expected << ", got " << actual << '\n';
    return 1;
}

int expect_absent(const qus::ArenaMemoryStats& stats, std::string_view label) {
    int failures = 0;
    failures += !stats.present ? 0 : fail(std::string(label) + " present");
    failures += expect_size(stats.capacity_bytes, 0, std::string(label) + " capacity");
    failures += expect_size(stats.used_bytes, 0, std::string(label) + " used");
    failures += expect_size(stats.peak_used_bytes, 0, std::string(label) + " peak");
    return failures;
}

int expect_present(const qus::ArenaMemoryStats& stats, std::string_view label,
                   bool require_used) {
    int failures = 0;
    failures += stats.present ? 0 : fail(std::string(label) + " absent");
    failures += stats.capacity_bytes > 0 ? 0 : fail(std::string(label) + " zero capacity");
    if (require_used) {
        failures += stats.used_bytes > 0 ? 0 : fail(std::string(label) + " zero used");
    }
    failures += stats.peak_used_bytes >= stats.used_bytes
                    ? 0
                    : fail(std::string(label) + " peak smaller than used");
    return failures;
}

int expect_peak_reset(const qus::ArenaMemoryStats& stats, std::string_view label) {
    if (!stats.present) { return fail(std::string(label) + " absent after reset"); }
    return stats.peak_used_bytes == stats.used_bytes
               ? 0
               : fail(std::string(label) + " peak was not reset to used");
}
